            <a href="" onclick="">Edit</a>
            <drop>
                <a href="">Stuff</a>
                <down>
                    <a href="" onclick="">Link 1</a>
                    <a href="" onclick="">Link 2</a>
                    <a href="" onclick="">Link 3</a>
                </down>
            </drop>
            <a href="" onclick="">Articles</a>
            <drop>
                <a href="">More</a>
                <down>
                    <a href="" onclick="">Link 1</a>
                    <a href="" onclick="">Link 2</a>
                    <a href="" onclick="">Link 3</a>
                </down>
            </drop>
        </nav>
    </div>

The logic behind the css is based on the https://www.w3schools.com/ article about hover dropdown.
