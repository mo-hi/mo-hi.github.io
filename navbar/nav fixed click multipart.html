nav.css

(with fixed position, click behavior and three sections)

The navbar of this page is an enhancement of nav with fixed position to the top and with the menu dropping down via onclick behavior instead of hover. In also comes with three nav sections, a left section, a right section and a middle section, whch can be used to display information to the user. It is created with the following html: 

        <nav class="w-60">
            <a href="" onclick="">Home</a>
            <a href="" onclick="">Edit</a>
            <drop>
                <a onclick="toggleDown(this)">Stuff</a>
                <down>
                    <a href="" onclick="">Link 1</a>
                    <a href="" onclick="">Link 2</a>
                    <a href="" onclick="">Link 3</a>
                </down>
            </drop>
        <nav>
            <info>
                Hallo
            </info>
        </nav>
        <nav class="w-40 flexR">
            <drop>
                <a onclick="toggleDown(this)">Things</a>
                <down>
                    <a href="" onclick="">Link 1</a>
                    <a href="" onclick="">Link 2</a>
                    <a href="" onclick="">Link 3</a>
                </down>
            </drop>
            <a href="" onclick="">Info</a>
        </nav>
