The navbar of this page is an enhancement of nav with fixed position to the top and with the menu dropping down via onclick behavior instead of hover. In also comes with three nav sections, a left section, a right section and a middle section, whch can be used to display information to the user. It is created with the following

CSS

Optionally you can add your custom function via NAV_AddGenericClickFunction, that will be called when the user clicks on a dropdown menu item. The function will not be called when the user clicks on a menu item without dropdown or on a dropdown child. The function will be assigned to the dropdown element via addEventListener, so you can use the event parameter to get the clicked element. See how the name of the clicked menu item is displayed in the info section.
