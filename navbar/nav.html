nav.css

The navbar of this page for example is created with the following html. 

    <nav>
        <a href="" onclick="">Home</a>
        <a href="" onclick="">Edit</a>
        <drop>
            <a href="">Stuff</a>
            <down>
                <a href="" onclick="">Link 1</a>
                <a href="" onclick="">Link 2</a>
                <a href="" onclick="">Link 3</a>
            </down>
        </drop>
        <a href="" onclick="">Articles</a>
        <drop>
            <a href="">More</a>
            <down>
                <a href="" onclick="">Link 1</a>
                <a href="" onclick="">Link 2</a>
                <a href="" onclick="">Link 3</a>
            </down>
        </drop>
    </nav>

⚠ Be aware that this navbar not 'fixed', i. e. if you scroll down on your page, then the navbar will scroll up with the rest of the page. If you want the navbar to be fixed, then use the nav fix position

The logic behind the css is based on the https://www.w3schools.com/ article about hover dropdown.
