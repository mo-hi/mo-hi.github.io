nav.css as sidebar

In its basic form, the only difference to make a navbar to a sidebar is to wrap the html inside a <ul> tag (instead of <nav>) and to use <li> tags (instead of <a>) for the sidebar items. Hence The sidear is intended to call functions and not intended to be used for links. Here's an example sidebar:
