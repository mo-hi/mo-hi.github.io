nav.css

(with fixed position and click behavior)

The navbar of this page is an enhancement of nav with fixed position to the top and with the menu dropping down via onclick behavior instead of hover. It is created with the following html: 

    <div class="position-fixed fullWidth">
        <nav>
            <a href="" onclick="">Home</a>
            <a href="" onclick="">Edit</a>
            <drop>
                <a onclick="toggleDown(this)">Stuff</a>
                <down>
                    <a href="" onclick="">Link 1</a>
                    <a href="" onclick="">Link 2</a>
                    <a href="" onclick="">Link 3</a>
                </down>
            </drop>
            <a href="" onclick="">Articles</a>
            <drop>
                <a onclick="toggleDown(this)">More</a>
                <down>
                    <a href="" onclick="">Link 1</a>
                    <a href="" onclick="">Link 2</a>
                    <a href="" onclick="">Link 3</a>
                </down>
            </drop>
        </nav>
    </div>
