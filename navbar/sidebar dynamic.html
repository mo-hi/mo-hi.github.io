nav.css as sidebar with dynamic content

In its basic form, the only difference to make a navbar to a sidebar is to wrap the html inside a <ul> tag (instead of <nav>) and to use <li> tags (instead of <a>) for the sidebar items. Hence The sidear is intended to call functions and not intended to be used for links.

In this version, the sidebar is created based on data. In your html project you have to call nav_AutoFillSidebar("id-sidebar", sidebarData) when the page loads with two parameters: The target divID (in This example'id-sidebar') and the actual data, from which the sidebar will be build (in this example 'sidebarData'). For example: 

                <script>
                    nav_AutoFillSidebar("id-sidebar", sidebarData)
                </script>
