nav.css

nav.css let's you create navbars / menubars on top of your html page in a intuitive way (at least that's what I aimed for). Div tag classes are not used, instead made-up html elements, like 'drop' and 'down' are used. No javascript, pure css.

In combination with position:fixed in a wrapping element, the navbar becomes fixed without losing its functionalty, i. e. it will keep its position and be visible even when you scroll down the page. Still pure css

The above navbar is created with just the following html:

    <nav>
        <a href="" onclick="">Home</a>
        <a href="" onclick="">Edit</a>
        <drop>
            <a href="">Stuff</a>
            <down>
                <a href="" onclick="">Link 1</a>
                <a href="" onclick="">Link 2</a>
                <a href="" onclick="">Link 3</a>
            </down>
        </drop>
        <a href="" onclick="">Articles</a>
        <drop>
            <a href="">More</a>
            <down>
                <a href="" onclick="">Link 1</a>
                <a href="" onclick="">Link 2</a>
                <a href="" onclick="">Link 3</a>
            </down>
        </drop>
    </nav>

Put inside a wrapper with a made-up tag <fixed>, defined in fixed.css

    <fixed class="resp">
        <nav>
            ...
        </nav>
    </fixed>

To use this way of defining a navbar / menubar, just insert the folllwoing css in your html page

    <link rel="stylesheet" href="css/nav.css">
    <link rel="stylesheet" href="css/YourColors.css">
    <link rel="stylesheet" href="css/YourPositionAndWidth.css">

Together with your custom css for fixed position and width.

Done. There's nothing more to do. But you may want to have your own color scheme.

If you want to put your custom color scheme, then create your own css based on navColors and place if below the nav.css. For example: 

    <link rel="stylesheet" href="css/nav.css">
    <link rel="stylesheet" href="../fixed.css"">
