bJS

This is B(asis) JS. A javascript class providing new functions you don't want to miss

To use bJS in your project, just add the following script reference to your project:

<script src="https://mo-hi.github.io/bjs/js/b.js"> </script>

That's it. There's nothing more to do. Enjoy! Link to github repo

[...].depth returns the max depth of an array. This function uses recursion. Per default recursions are limited to 9, hence it will return a max value of 9 even if the depth of the array is higher. You can pass a new limit for the max recursion level. let b = [1, 2, 3].depth() // b = 1
let b = [1, [2], 3].depth() // b = 2
let b = [1, [2, [3]]].depth() // b = 3
