b.CSS

To use b.CSS in your project, just add the following css reference to your project. in case you also want to use the icons, add the svg.js reference as well.

(-html-)
--(-head-)
----(-link rel="stylesheet" href="https://mo-hi.github.io/bjs/css/b.css"-)
--(-/head-)
--(-body-)
----(-link rel="stylesheet" href="https://mo-hi.github.io/js/svg.js"-) // only if you want to use the icons
--(-/body-)
(-/html-)

Usage Example: Button Minimum HTML

Icons

The code for icons are stored in

(-link rel="stylesheet" href="https://mo-hi.github.io/js/svg.js"-)

You can either
- directly use the svg code form the file (CopyPaste)
- call the svg code in javascript via the dictionary vaule (FromFile_svg_js['svg-icon-edit-24'])
- use the Auto_Fill function from bJS to fill the svg code into the html via {{ svg-icon-edit-24 }}
